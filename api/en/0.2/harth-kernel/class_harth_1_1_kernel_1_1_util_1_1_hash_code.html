Detailed Description

FNV-1a Hashing function.

Not suitable for security applications though. See http://en.wikipedia.org/wiki/Fowler–Noll–Vo_hash_function

NOTE: HashCode().Mix(block1).Mix(block2) will be the same for any contiguous block1 and block2 split from a larger block of memory/string data. For example:

HashCode().Mix("he").Mix("llo") == HashCode().Mix("h").Mix("ello").

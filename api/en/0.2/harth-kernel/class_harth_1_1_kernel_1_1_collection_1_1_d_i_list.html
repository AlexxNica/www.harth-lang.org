Detailed Description

template<typename T, size_t Offset = T::GetListNodeOffset()>
class Harth::Kernel::Collection::DIList< T, Offset >

D*oubly linked *I*ntrusive *List

Rules:

  • DIList must be destructed empty, so Remove() or use RemoveAll() explicitly first.
  • DIListNode must be destructed after being Remove() or RemoveAll() from list.
  • Element type using DIListNode should use HK_DECLARE_DILIST_NODE() which: – Embeds a private DIListNode field in the struct or class. – Provides the required static constexpr T::GetListNodeOffset() function.

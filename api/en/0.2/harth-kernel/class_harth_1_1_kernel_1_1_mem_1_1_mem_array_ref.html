Detailed Description

template<typename T>
class Harth::Kernel::Mem::MemArrayRef< T >

Describes a reference to an array of elements stored in contiguous memory.

Similar to MemRef. Elements are typically of the given type T, and are sized sizeof(T). Optionally the actual elementByteSize can be passed, where type T is smaller than the actual element (useful for storing pointers to base classes).

Note
This is an immutable data structure of small size, it should typically be passed and returned as a value type (not as a reference type).
